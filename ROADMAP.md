# Roadmap

## Free Beta Goals

- Reliable install/uninstall on clean Windows 10/11 machines.
- Fewer false positives.
- Better missed-detection reports.
- Clear recovery-code onboarding.
- Stable service/agent behavior.
- GitHub Releases with checksums.

## Before Charging

- Resolve remaining PyQt distribution licensing for the lockout/uninstall guard surfaces: buy a commercial PyQt license or port those surfaces to PySide6/LGPL.
- 20+ beta downloads.
- 5+ confirmed successful installs.
- 3+ users who use it for at least a week.
- No critical lockout/uninstall bugs open.
- README and recovery flow understandable by strangers.
