# Recovery Code

The recovery code is your emergency path back into Brake.

Brake shows it once on first launch. You will not see the same code again.

## What it can do

The recovery code can:

- reset your protection password immediately
- start a 10-minute emergency cooldown before Brake turns protection off
- help you recover during an active commitment

## Why there is a cooldown

Without a cooldown, the recovery code would become an easy bypass. The delay gives you time to step away from the impulse before protection turns off.

## Where to keep it

Best options:

- write it on paper
- take a photo on your phone
- give it to someone you trust

Avoid saving it in a text file, notes app, password manager, or browser tab on the same computer if you want strong commitment.

## Strongest commitment option

You can choose not to copy the recovery code. This makes Brake harder to bypass, but it also means a forgotten password may require a full reset.
