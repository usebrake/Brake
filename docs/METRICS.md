# Passive launch metrics

Brake does not collect analytics from the desktop app.

For launch checks, use passive GitHub numbers:

- **Stars:** open `https://github.com/usebrake/Brake` and check the star count near the repo title.
- **Release downloads:** open the latest GitHub Release and check the download count beside each asset.
- **Traffic:** open the repo, then go to **Insights > Traffic**. GitHub only keeps this data for a limited time.

These numbers come from GitHub. They do not require any code in Brake and do not add telemetry to the app.
