# Brake Brand Copy

## Product promise

Brake adds a private, local pause between impulse and action.

## Primary tagline

Local screen accountability.

## Alternate lines

- Local accountability for moments that matter.
- A quieter way to stay accountable.
- Serious friction without surveillance.
- Built for explicit content. Designed for privacy.

## Voice

Calm, direct, and non-judgmental.

Use plain words. Avoid shame, hype, therapy-speak, and moral grandstanding. Brake should feel like serious software made for a sensitive problem.

## Words to prefer

- explicit content
- protection
- commitment
- recovery code
- local checks
- full lockout

## Words to avoid in product UI

- addiction cure
- porn blocker claims
- guaranteed
- impossible to bypass
- shame-based language
