# Legacy PyQt Control Panel

Archived 2026-06-18.

These files were the old PyQt control-panel shell before the Electron control panel became the active Brake UI. They are kept here as a backup/reference only.

Do not delete the remaining active PyQt files in `brake/gui`, `brake/lockout`, or `brake/uninstall_guard.py` yet. The current app still uses PyQt for the lockout window, uninstall guard, password dialogs, and shared assets.
